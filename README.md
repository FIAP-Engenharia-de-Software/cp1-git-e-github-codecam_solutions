# 🌐 Case-Social-Connect

## Resumo do Projeto

O **Social Connect** é uma plataforma web voltada para conectar pessoas com interesses em comum, promovendo networking pessoal e profissional. A proposta é oferecer um ambiente digital onde usuários possam expandir suas conexões, trocar experiências e crescer em conjunto.

A aplicação apresenta uma interface simples e intuitiva, com seções institucionais como apresentação da empresa, equipe e formas de contato.
